# Notebooks

Suggested notebook order after downloading data:

1. `01_data_audit.ipynb`: schema, missingness, and target validation.
2. `02_subscription_churn_eda.ipynb`: subscription usage, churn drivers, and customer segments.
3. `03_ott_dropoff_eda.ipynb`: episode retention, season fatigue, and platform/genre risk.
4. `04_baseline_modeling.ipynb`: reproducible model training, evaluation, and business interpretation.

Keep exploratory work here and move reusable code into `src/churn_retention/`.
